# Cascade Technical Whitepaper — Build System

## Quick Start

```bash
# Build PDF
make -f docs/paper/_build/Makefile pdf

# Build draft (with watermark + line numbers)
make -f docs/paper/_build/Makefile draft

# Word count
make -f docs/paper/_build/Makefile wordcount
```

## Dependencies

### macOS

```bash
brew install pandoc pandoc-crossref
# TeXLive via MacTeX: https://www.tug.org/mactex/
```

### lxplus (CERN)

```bash
# pandoc — download binary release
wget https://github.com/jgm/pandoc/releases/download/3.1.11/pandoc-3.1.11-linux-amd64.tar.gz
tar xzf pandoc-3.1.11-linux-amd64.tar.gz
export PATH=$PWD/pandoc-3.1.11/bin:$PATH

# pandoc-crossref — download matching release
wget https://github.com/lierdakil/pandoc-crossref/releases/download/v0.3.17.0/pandoc-crossref-Linux.tar.xz
tar xf pandoc-crossref-Linux.tar.xz
chmod +x pandoc-crossref
export PATH=$PWD:$PATH

# TeXLive via CVMFS
export PATH=/cvmfs/sft.cern.ch/lcg/external/texlive/2023/bin/x86_64-linux:$PATH
```

## Pipeline Architecture

```
docs/paper/*.md  ──┐
                   ├──▶ pandoc ──▶ latexmk ──▶ PDF
00-metadata.yaml ──┤     │           │
                   │     ├── pandoc-crossref (Sec./Fig./Eq. refs)
_build/            │     ├── citeproc ([@bibkey] citations)
├── templates/     │     └── template.tex (LaTeX layout)
│   └── arxiv-clean.tex
├── filters/
│   └── crossref-meta.yaml
├── references.bib
└── figures/
```

**Critical:** `--filter=pandoc-crossref` must come **before** `--citeproc` in the pandoc command.
pandoc-crossref processes `[@sec:xxx]` references first, then citeproc handles `[@bibkey]` citations.

## Template Switching

See `templates/registry.yaml` for available journal templates. To switch:

1. Download the journal's `.cls` and `.bst` files into `templates/`
2. Update `TEMPLATE` in `Makefile` to point to the new template
3. Adjust template-specific pandoc variables as needed

## Files NOT Managed by cas-texwriter

The entire `_build/` directory is **invisible to the Agent**:
- `Makefile` — human-maintained
- `templates/` — human-maintained  
- `filters/` — human-maintained
- `references.bib` — human-maintained
- `figures/` — human-maintained
- `.pillar_baselines` — auto-generated by `/cas-texwriter init`, read-only thereafter
- `.pre_patch_backup/` — auto-created by Step 7, auto-cleaned by Step 8
